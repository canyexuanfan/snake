# 灵动贪吃蛇（SNAKE.IO）

一个基于 HTML5 Canvas + 原生 JavaScript + CSS3 的贪吃蛇小游戏，内置 **7 个关卡、4 档难度、4 套主题** 和道具系统，支持桌面键盘与移动端触控。

🎮 **在线体验**：<https://snake.hnwen17.top>

---

## 功能特性

- **7 个关卡**：从草地到岩浆，墙壁样式覆盖边框 / 十字 / 网格 / 螺旋 / 迷宫 / 随机
- **4 档难度**：简单 / 普通 / 困难 / 极速，随时切换立即生效
- **4 套主题**：草地、沙漠、雪地、岩浆，每个主题独立配色
- **道具系统**：
  - 🍎 普通苹果 `+10`
  - ⭐ 金色道具 `+50`（加速 5 秒）
  - 🔷 减速道具 `+30`（减速 3 秒）
  - 💣 炸弹 `-20`（蛇身缩短 3 格）
- **多种控制方式**：方向键 + 移动端滑动手势 + 虚拟方向键（D-Pad）
- **Web Audio API 实时合成音效**：吃食物 / 游戏结束 / 升级通关，无需音频文件
- **本地数据持久化**：最高分、累计场次、最长蛇身等通过 `localStorage` 保存
- **响应式布局**：桌面三栏布局，移动端单列 + 虚拟方向键

## 技术栈

- HTML5 Canvas（20×20 网格，400×400 像素）
- 原生 JavaScript（无外部框架）
- CSS3（响应式 + 主题切换）
- Node.js HTTP Server（本地开发，端口 8000）

## 本地启动

```bash
# 安装依赖（当前无外部依赖）
npm install

# 启动本地服务器（默认端口 8000）
npm start
# 或直接运行
node server.js
```

访问 <http://127.0.0.1:8000> 即可开始游戏。

## 控制方式

| 平台 | 操作 |
|------|------|
| 桌面 | ↑ ↓ ← → 方向键控制方向；游戏未运行时按任意方向键自动开始 |
| 移动端 | 页面任意位置滑动手势；或使用自动显示的虚拟方向键 |

## 项目结构

```text
.
├── index.html       # 三栏式响应式布局入口
├── script.js        # 游戏主逻辑（关卡 / 道具 / 主题 / 音效）
├── style.css        # 主题与响应式样式
├── server.js        # Node.js 静态文件服务器
├── vercel.json      # Vercel 部署配置
├── start.bat        # Windows 一键启动脚本
└── 需求文档.md      # 完整需求文档
```
